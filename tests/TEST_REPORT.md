1. **Файл → Загрузить пример из методики** — проверить загрузку эталонного проекта
2. **Рассчитать** — проверить отображение результатов и диаграмм
3. **Экспорт в Word/Excel** — проверить открытие сгенерированных файлов
4. **Сохранение/загрузка проекта** — полный цикл
5. **Каталог функций** — добавление компонента, выбор функции из каталога
6. **Коэффициенты** — изменение значений и перерасчёт
